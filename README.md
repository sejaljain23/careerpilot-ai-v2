## 🌟 Project Overview

CareerPilot AI is an AI-powered career guidance platform that helps students and professionals improve their careers using Google's latest AI technologies.

The platform combines **Google Gemini**, **Google ADK**, **FastMCP**, and a **Multi-Agent Architecture** to provide intelligent resume analysis, career guidance, and interview preparation.

---

## ✨ Features

- 📄 AI Resume Analysis
- 📊 ATS Score Generation
- 💼 Personalized Career Guidance
- 🎤 Interview Question Generator
- 🧠 Session Memory
- 📥 PDF Report Download
- 🤖 Google ADK Integration
- 🔌 FastMCP Integration
- 🚀 Multi-Agent Architecture
- 🌍 Live Deployment (Render)
